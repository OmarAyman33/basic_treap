\section*{Treap Implementation}
A minimal templated C++ Treap supporting fast insertion, deletion, search, splitting, and range queries.  
Balanced using randomized priorities while maintaining BST ordering.

\subsection*{Features}
\begin{itemize}
    \item Template-based Treap structure
    \item Insert, erase, search
    \item Split and rangeQuery
    \item Copy constructor and inorder traversal
\end{itemize}
